# Academic use

This repository is the source code and evaluation artifacts for a master's thesis (TFM) on RAG over EU legal text (MultiEURLEX).

- **Authors:** Adriana Ibarra, Michael Wetzel, Mauricio Orozco, Dylan Estrada 
- **Dataset:** [MultiEURLEX](https://huggingface.co/datasets/coastalcph/multi_eurlex) (accept Hugging Face terms; use `HF_TOKEN` for download).
- **Third-party models:** Ollama embedding/chat models, Hugging Face cross-encoder checkpoints, optional Google Gemini API for the webapp and Ragas judge.

Do not commit API keys (`.env` is gitignored). See [README.md](../README.md) for reproduction steps and [REPRODUCING_RESULTS.md](REPRODUCING_RESULTS.md) for committed result files.
